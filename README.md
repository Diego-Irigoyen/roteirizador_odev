# 📍 Roteirizador ODev

Projeto desenvolvido para otimização de rotas de atendimento técnico, utilizando **Python** e a **Google Maps API**. 🚀

## ✨ Funcionalidades
* Otimização de trajetos por proximidade (Algoritmo do Vizinho Mais Próximo).
* Leitura dinâmica de destinos via arquivo CSV.
* Foco em segurança de dados (uso de variáveis de ambiente e .gitignore).

## 🛠️ Tecnologias
* Python 3.13
* Google Maps Distance Matrix API
* Pandas & Python-Dotenv

---
*Desenvolvido por sentir a necessidade de uma ferramenta gratuita que pudesse organizar meus roteiros de forma simples.*
